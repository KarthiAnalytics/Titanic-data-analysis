# Titanic Data Analysis & Excel Dashboard

## Project Overview
This project focuses on cleaning and analyzing the Titanic dataset using Python and visualizing insights using Excel.

## Tools Used
- Python (Pandas, NumPy)
- Excel (Pivot Tables, Charts, KPIs, Slicers)

## Key Steps
- Cleaned missing and inconsistent data
- Created new features such as Family Size, IsAlone, Age Group, and Fare Group
- Exported cleaned data to Excel
- Built an interactive Excel dashboard to analyze survival patterns

## Key Insights
- Higher survival rate among females
- First-class passengers had better survival chances
- Passengers traveling alone had lower survival rates

## Files
- Titanic_Cleaning.ipynb – Data cleaning and preparation
- Titanic_Dashboard.xlsx – Excel dashboard
